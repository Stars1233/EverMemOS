# Data

This directory holds evaluation datasets. It is not tracked by git.

Download from [HuggingFace](https://huggingface.co/datasets/EverMind-AI/EvoAgentBench):

```bash
huggingface-cli download EverMind-AI/EvoAgentBench --repo-type dataset --local-dir ./data
```

See the main [README](../README.md) for details.
